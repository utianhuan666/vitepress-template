# 安装

## 环境要求

- Node.js 16+
- Vue 3.0+

## 包管理器安装

::: code-group

```bash [npm]
npm install pro-components
```

```bash [yarn]
yarn add pro-components
```

```bash [pnpm]
pnpm add pro-components
```

:::

## CDN 引入

```html
<script src="https://unpkg.com/pro-components"></script>
